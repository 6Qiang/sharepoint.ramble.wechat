# New-SPWeb
	作者：jingnansu

在现有网站集中创建新的网站.

## 参数
> - "Url": 创建新网站的地址.
> - "Language": 新网站的语言.
> - "Template": 新网站的模板.
> - "AddToQuickLaunch": 将新网站增加到"快速启动"中.
> - "AddToTopNav": 将新网站增加到"顶级导航栏"中.
> - "UseParentTopNav": 新网站将于父网站使用相同的顶级导航栏.

## 例子
> - `New-SPWeb http://spsite/subweb -Template "STS#0"`
